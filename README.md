## Project Overview

This project was completed as part of the PwC Data Science Graduate Program Assessment.

The objective was to analyse a gaming review dataset and develop a data-driven solution that helps GameVault Publishing identify games with strong acquisition and licensing potential.

GameVault currently relies on manual review processes and simple metrics such as ratings and review counts. The proposed solution aims to provide a more scalable and consistent approach by incorporating player engagement metrics, review sentiment, and game performance indicators into a centralised decision-support system.

---

## Business Problem

GameVault Publishing faces three primary challenges:

* Difficulty distinguishing genuinely successful games from short-term trends.
* Manual and inconsistent evaluation processes.
* Lack of a centralised platform for comparing potential acquisition targets.

The goal of this project is to provide a framework that enables GameVault to prioritise games with strong long-term commercial potential.

---

## Approach

The project was completed using the following process:

### 1. Data Analysis

* Explored the gaming review dataset.
* Performed data cleaning and preprocessing.
* Investigated player engagement metrics.
* Analysed review trends and sentiment indicators.
* Identified patterns associated with successful games.

### 2. Prioritisation Framework

A prioritisation methodology was developed to rank games according to their acquisition potential.

The framework considers:

* Player engagement
* Review sentiment
* Review volume
* Overall player reception
* Indicators of sustained appeal

### 3. Dashboard / Solution Design

A dashboard was developed to provide:

* Centralised game comparison
* Ranking of acquisition opportunities
* Key performance indicators
* Interactive filtering and exploration capabilities

The dashboard is intended to support faster and more consistent decision-making across the GameVault team.

---

## Key Findings

Some of the most important findings from the analysis include:

* Highly rated games are not always the strongest acquisition candidates.
* Player engagement provides additional insight beyond review scores alone.
* Sentiment patterns can help identify games with strong long-term appeal.
* Combining multiple indicators produces a more reliable prioritisation strategy than relying on a single metric.
